⚙️ Conexión con GitHub

Para guardar tus decisiones necesitas un token personal de GitHub con permiso de escritura en este repositorio. Se guarda solo en tu navegador (localStorage), nunca se sube a ningún lado.

  1. Andá a GitHub → Fine-grained tokens.
  2. Creá un token con acceso a este repo y permiso Contents: Read and write.
  3. Pegalo acá abajo y guardá.
